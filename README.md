# mini-dotfiles

## Install

```bash
$ git clone git@github.com:3tty0n/mini-dotfiles.git ~/.dotfiles
$ cd ~/.dotfiles
$ ./install.sh
```

## Minimal Install

```bash
$ git clone git@github.com:3tty0n/mini-dotfiles.git ~/.dotfiles
$ cd ~/.dotfiles
$ make build.mini
```
