# .dotfiles

## Install

```bash
git clone git@github.com:3tty0n/.dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.sh
```

## Minimal Install

```bash
git clone git@github.com:3tty0n/.dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install-mini.sh
```
