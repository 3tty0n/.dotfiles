# .dotfiles

## Install

### stable (use zplug)

For stability.

``` bash
git clone git@github.com:3tty0n/.dotfiles.git ~/.dotfiles && cd ~/.dotfiles
./install.sh --master
```

### develop (use zgen)

For speed.

```bash
git clone git@github.com:3tty0n/.dotfiles.git ~/.dotfiles && cd ~/.dotfiles
./install.sh --develop
```
